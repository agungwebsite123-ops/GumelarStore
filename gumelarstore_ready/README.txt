GumelarStore - Ready package (XAMPP)
==================================
Instruksi singkat instalasi:

1. Ekstrak folder 'gumelarstore' ke C:\xampp\htdocs\gumelarstore
2. Jalankan Apache & MySQL di XAMPP Control Panel
3. Jika database belum ada: buka http://localhost/gumelarstore/setup.php (setup menggunakan IF NOT EXISTS so safe)
   - Script akan mengimpor skema DB jika perlu dan membuat admin default Agung/Agung123 jika belum ada.
   - Setelah selesai: HAPUS file setup.php demi keamanan
4. Buka situs: http://localhost/gumelarstore/
   Admin panel: http://localhost/gumelarstore/index.php?page=admin
   - Export laporan penjualan: Admin -> Export Sales (CSV)
